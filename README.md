# Student Marks Analyzer

This project analyzes student marks using Python.

## Features
- Takes marks as input
- Calculates average marks
- Finds highest and lowest marks
- Assigns grade based on performance

## Grading System
- 90 and above: Grade A
- 75 to 89: Grade B
- 50 to 74: Grade C
- Below 50: Fail

## How to run
1. Enter number of subjects
2. Enter marks for each subject
3. View analysis results

## Tech Used
- Python
